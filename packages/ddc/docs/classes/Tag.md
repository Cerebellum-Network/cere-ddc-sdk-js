# Class: Tag

The `Tag` class represents a DAG Node tag.

**`Example`**

```typescript
const key = 'exampleKey';
const value = 'exampleValue';
const tag = new Tag(key, value);

console.log(tag);
```

## Properties

### key

• **key**: `string`

The key of the tag.

___

### value

• **value**: `string`

The value of the tag.

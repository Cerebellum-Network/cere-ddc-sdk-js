# @cere-ddc-sdk/ddc

The package provides API for interacting with DDC Storage nodes.

# Installation

Using `NPM`:

```bash
npm install @cere-ddc-sdk/ddc --save
```

Using `yarn`:

```bash
yarn add @cere-ddc-sdk/ddc
```
